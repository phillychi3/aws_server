使用方式
安裝下面一大堆東西


```bash
pip install flask
pip install grequests
pip install requests
pip install yaspin
pip install youtube-lv
```

目前有兩個模式 分別是使用一個server的base64 mod 以及使用兩個server的 neko mod  

使用一個server:
>把server點開 或是使用python ./server  
>然後點開client 按照上面的提示輸入


使用兩個server:
>把server點開  
>把nekoserver點開  
>也是打開client 然後輸入上面的東西
